# DESIOLE Kitchen Site Plan

## Brand Positioning

DESIOLE Kitchen is positioned as a factory-style B2B kitchen tools supplier in China.

Primary positioning statement:

Wholesale Kitchen Tools Supplier In China

The website should communicate factory-direct supply, low MOQ, custom branding, private label packaging, OEM/ODM manufacturing, Amazon FBA support and export-ready quality control.

## Target Buyers

- Importers and wholesale distributors sourcing kitchen tools from China.
- Amazon sellers and e-commerce operators needing product preparation support.
- Retail brands looking for custom logo or private label packaging.
- Promotional product buyers needing low MOQ customization.
- Chain stores and regional buyers comparing factory-direct product ranges.

## WooCommerce Catalog Mode Strategy

WooCommerce should be used as a catalog and inquiry system, not a retail checkout store.

Rules:

- Hide prices globally.
- Remove Add to Cart buttons.
- Disable purchasing.
- Hide quantity selectors.
- Hide cart and checkout links.
- Redirect cart and checkout pages to `/request-quote/`.
- Product cards should use Inquiry or Request Quote buttons.
- Product detail pages should use Request Quote with product query parameter.

## Product Detail Page Strategy

Single product pages should focus on B2B decision-making.

Expected content:

- Product image gallery
- Product name
- Short B2B description
- Key features
- Specification table
- Custom options
- Packaging options
- MOQ information
- Sample lead time
- Bulk production time
- Amazon FBA support note
- Request Quote button
- Related products

The template should support product meta fields when available and show sensible fallback text when fields are empty.
