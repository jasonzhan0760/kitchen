# Collaboration Workspace

This folder is the handoff workspace for planning and implementation.

## Source Of Truth

Priority order:

1. `README.md` in the repository root
2. Files in this `docs/` folder
3. Later implementation notes and commit history

If any document conflicts with `README.md`, keep `README.md` as the final rule and record the conflict in `docs/decision-log.md`.

## How To Work With ChatGPT

Ask ChatGPT to place or update its website plan in these files:

- `site-plan.md`
- `page-structure.md`
- `content-plan.md`
- `design-system.md`
- `woocommerce-rules.md`
- `implementation-brief.md`

ChatGPT should avoid changing root project rules unless explicitly requested.

## How Codex Will Use This Folder

Codex will read these documents before implementation, convert the plan into files and tasks, and list changed files after each task.
