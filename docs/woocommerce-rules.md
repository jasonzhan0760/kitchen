# WooCommerce Rules

## Catalog Mode

- Hide prices globally.
- Remove Add to Cart buttons.
- Hide cart and checkout links.
- Replace purchase actions with Request Quote CTAs.

## Product Archives

Define product card layout, category filters, and inquiry buttons.

## Product Detail Pages

Define product fields, quote CTA placement, and related products.

## Request Quote Links

Use `/request-quote/`.

When possible, pass the product name:

```text
/request-quote/?product=PRODUCT_NAME
```

## Admin Editing Rules

Document what should be editable in WordPress and WooCommerce admin.
